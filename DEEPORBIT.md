# DEEP ORBIT

A Star Reach (1994) inspired hybrid strategy/arcade space game built in SpriteKit.

## Vision
Fly between planets on a galaxy map, build economies, raise fleets, and fight top-down arcade combat encounters to conquer the galaxy. NASApunk aesthetic throughout.

## Milestones

- [ ] **M0: Foundation** — Extract shared code, data models, theme
- [ ] **M1: Galaxy Map** — Navigate between planets, fuel system
- [ ] **M2: Planet Management** — Build mines, factories, spaceports, biospheres
- [ ] **M3: Combat Integration** — Wave-based encounters, capture planets *(MVP)*
- [ ] **M4: AI Factions** — 3 alien races expand and compete
- [ ] **M5: Fleet System** — Build ships, fleet affects combat
- [ ] **M6: Win/Lose + Polish** — Save/load, end states, campaign flow *(1.0)*
- [ ] **M7: Phantom Mode** — Strategic overview, diplomacy *(stretch)*
