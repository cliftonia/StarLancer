# DEEP ORBIT

A Star Reach (1994) inspired hybrid strategy/arcade space game built in SpriteKit.

## Vision
Fly between planets on a galaxy map, build economies, raise fleets, and fight top-down arcade combat encounters to conquer the galaxy. NASApunk aesthetic throughout.

## Milestones

- [x] **M0: Foundation** — Extract shared code, data models, theme
- [x] **M1: Galaxy Map** — Navigate between planets, fuel system
- [x] **M2: Planet Management** — Build mines, factories, spaceports, biospheres
- [x] **M3: Combat Integration** — Wave-based encounters, capture planets *(MVP)*
